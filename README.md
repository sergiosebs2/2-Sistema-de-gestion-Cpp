# 🗂️ Sistema de Gestión en C++

Proyecto de sistema de gestión desarrollado en C++ con lectura y escritura de ficheros. Uno de mis primeros proyectos aplicando conceptos de programación orientada a objetos y manejo de archivos.

## 🛠️ Tecnologías

![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white)

## 📌 Sobre el proyecto

Sistema de gestión que permite administrar datos mediante la persistencia en ficheros. El proyecto aplica conceptos como:

- Lectura y escritura de archivos
- Estructuras y clases en C++
- Gestión de datos en memoria
- Menú interactivo por consola
